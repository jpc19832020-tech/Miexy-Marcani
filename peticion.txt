GLOSARIO RÁPIDO

GitHub Pages: hosting estático gratuito desde un repo.

GitHub Actions: automatiza pasos (build/deploy) cuando hay un push.

Play CDN (Tailwind): forma de usar Tailwind sin proceso de build.

Autoplay: avance automático del carrusel cada X segundos.

Fallback: alternativa cuando una función (Web Share) no está disponible.
